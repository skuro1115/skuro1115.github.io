# 技術ブログ収益化・応援導線メモ

## 前提

将来的な機能で現状は実装は見送り

## 方針

技術ブログでは、記事そのものの価値を主役にする。

応援・収益化の導線は前面に出しすぎず、記事末尾やプロフィールページに控えめに配置する。

現時点では、単純な寄付ボタンのみを設置対象とする。

ほしい物リストと購入済み商品のアフィリエイトは、将来的な機能として検討に留める。

## 現行機能

### 100円寄付ボタン

記事が役に立った読者が、軽く応援できる導線として設置する。

金額は小さくし、支援の圧を出さない。

主な目的は収益化ではなく、記事作成・検証・開発活動の継続を支える小さな応援導線とする。

#### 表示文言案

```text
この記事が役に立った場合は、100円で応援できます。
今後の記事作成・検証環境の維持に使います。
```

または、

```text
記事が参考になったら、100円サポートで応援できます。
無理のない範囲での応援用です。
```

## 将来機能

### ほしい物リスト

研究・開発・読書・作業環境改善に関係するものをまとめる。

「買ってください」という見せ方ではなく、自分の関心領域や活動方針が伝わるプロフィール拡張として使う。

#### 想定カテゴリ

- 読みたい技術書
- 研究・開発に使う書籍
- 作業中の飲料
- デスク環境改善グッズ
- VR / XR 関連デバイス
- 開発効率を上げる周辺機器

#### 表示文言案

```text
研究・開発・読書・作業環境改善に関係するものをまとめています。
差し入れ用というより、自分の関心領域のメモも兼ねています。
```

---

### 購入済み商品のアフィリエイト

実際に購入・使用した書籍やデバイスを中心に紹介する。

読者にとっても価値がある形にするため、単なる商品リンクではなく、使用感・用途・良かった点・注意点を添える。

#### 想定カテゴリ

- 実際に使ってよかった開発道具
- 研究・開発を支えるデバイス
- 技術書
- 作業環境改善グッズ
- VR / XR 開発関連アクセサリ
- デスクワーク疲労を減らす道具

#### 表示文言案

```text
実際に購入・使用した書籍やデバイスを中心に紹介しています。
一部リンクにはアフィリエイトリンクを含み、購入により収益が発生する場合があります。
```

## 記事末尾の表示案

現時点では、寄付ボタンのみを表示する。

```text
この記事が役に立った場合は、100円で応援できます。
今後の記事作成・検証環境の維持に使います。
```

将来的に、ほしい物リストや購入済み商品の紹介を追加する場合は、以下のようにする。

```text
この記事が役に立った場合は、100円サポートで応援できます。

また、実際に使ってよかった本・デバイスや、研究・開発で気になっているものもまとめています。
```

---

## 注意点

- 技術記事の本文中に支援導線を入れすぎない。
- 記事冒頭には置かず、基本は記事末尾かプロフィールページに置く。
- 高額商品の押し出しは避ける。
- アフィリエイトを使う場合は、広告・アフィリエイトであることを明示する。
- 医薬品・サプリ・生活感が強すぎるものは公開リストに入れない。
- 「支援してください」ではなく「役に立ったら応援できます」程度の表現にする。

---

## 基本スタンス

技術ブログの主役は、技術記事・実装記録・検証結果・学習ログ。

応援導線は、活動継続のための補助的な仕組みとして扱う。

ほしい物リストやアフィリエイトは、単なる収益化ではなく、自分の関心・開発環境・研究活動の方向性が伝わる形で将来的に導入する。
